# System Alert Notifications

## Service Description

This phone number is used to send automated system notifications and
operational alerts to the owner of the system.

Messages are:
- Non-promotional
- Low volume
- Sent only to a single opted-in recipient (the system owner)

## Message Examples

- "Server health check failed"
- "Trading system alert triggered"
- "Process restarted successfully"
- "Monitoring warning detected"

## Opt-In Information

Messages are sent only to the system owner who has explicitly opted in
to receive these notifications.
